# How to build the UI

1. Install NodeJS <https://nodejs.org>
2. Enable Corepack `corepack enable`
3. Install dependencies `pnpm i`
4. Run dev (while editing) `pnpm run dev`
5. Build the HTML with `pnpm run build`
